06 — Motion

Quiet entrance.

The logo animates in two stages: the solid ellipse appears first, then the globe wireframe and wordmark fade in together with a slight upward drift. Total sequence under one second.

Storyboard
/* ──── Logo Entrance ──── */
0ms solid ellipse fades in (opacity 0 → 1, 400ms ease-out) 200ms wireframe + wordmark fade in (opacity 0 → 1, translateY 3px → 0, 500ms) 900ms sequence complete — content may enter
Timing Constants
fill
0ms
Ellipse appears immediately

200ms

Solid shape. Simple deceleration.

Usage

Always use both stages. The ellipse and detail are never shown simultaneously without the entrance. If animation isn't possible, show the full lockup static.

Keep it under a second. The logo entrance should never block content. 900ms total including the safety buffer before page content begins.

Play once. The animation triggers once per page load or scroll-into-view. Never loop.

CSS Reference
/* Stage 1: Solid ellipse */
.logo-fill {
  opacity: 0;
  transition: opacity 0.4s ease-out;
}
.logo-fill.anim-in {
  opacity: 1;
}

/* Stage 2: Globe wireframe + wordmark */
.logo-detail {
  opacity: 0;
  transform: translateY(3px);
  transition:
    opacity 0.5s cubic-bezier(0.25, 0.46, 0.45, 0.94),
    transform 0.5s cubic-bezier(0.25, 0.46, 0.45, 0.94);
}
.logo-detail.anim-in {
  opacity: 1;
  transform: translateY(0);
}
Apply anim-in to .logo-fill at 0ms, .logo-detail at 200ms
